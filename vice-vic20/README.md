# KULLS

The src directory contains a version of Kulls which operates within the
VICE VIC-20 emulator.


## Keyboard mapping

- A player can press no key to remain in the same position (i.e. stationary).
- A player can press one of the keys in the table below to move in the
  corresponding direction. E.g. Player 1 can press the 'A' key to move left.
- A player can press two of the keys in the table below to move diagonally.
  E.g. Player 1 can press the 'S' key and 'D' key simultaneously to move
  diagonally down and right.

PLAYER   | Up | Left | Down | Right
---------|----|------|------|-------
Player 1 |  W |    A |    S |     D
Player 2 |  O |    K |    [ |     ;
Player 3 |  T |    F |    G |     H
Player 4 |  J |    N |    M |     .


## Controller / joystick mapping

In Windows or Linux, you can install an operating system level app to
re-map gamepad joystick positions to keyboard keys.  For example:

- https://github.com/sezanzeb/input-remapper (e.g. available from MX Package Installer > MX Test Repo in MX Linux 23.3)
- https://github.com/antimicroX/antimicroX
